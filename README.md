### to run this application you must:
* clone the repository
```
git clone <this repository>
```
* install node.js
* open the folder and run on the terminal
```
npm install
npm install pm2 -g
```
* Run 
```
npm start
```
* monitor the application on console 
```
pm2 monit
```
